## 🇩🇪 Deutsch

### 📜 Projekt-Vision
Ein minimalistisches, schnelles Toolkit für den Spieltisch, optimiert für **D&D 5e (2014 & 2024)** sowie offizielle Erweiterungsbücher. Entwickelt, um den Spielfluss zu beschleunigen, indem komplexe Tabellen und Beschreibungen verschiedener Editionen sofort griffbereit sind.

### 🛠️ Werkzeuge & Module
| Modul | Beschreibung |
| :--- | :--- |
| **Grimorium** | Datenbank für Zaubersprüche (Legacy, 2024 & Erweiterungen). |
| **Meisterschaften** | Übersicht der *Weapon Masteries* und Kampf-Features. |
| **Arsenal** | Datenbank für Waffen, Rüstungen und magische Gegenstände. |
| **Zustände** | Schneller Check für Konditionen und allgemeine Spielregeln. |

## 🇺🇸 English

### 📜 Project Vision
A minimalist, high-speed toolkit for the gaming table, covering **D&D 5e (2014 & 2024)** and official expansion content. Built to keep the narrative moving by providing instant access to rules and specifics across different editions.

### 🛠️ Modules
| Module | Description |
| :--- | :--- |
| **Grimorium** | Searchable spell database (Legacy, 2024 & Expansions). |
| **Masteries** | Quick reference for *Weapon Masteries* and combat features. |
| **Arsenal** | Equipment database including base gear and magic items. |
| **Conditions** | Instant lookup for all conditions and core mechanics. |
