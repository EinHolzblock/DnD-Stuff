# Mitwirken am DnD Workshop ⚔️

Vielen Dank, dass du helfen möchtest, dieses Tool zu verbessern! Um die Qualität und Struktur des Projekts zu erhalten, beachte bitte folgende Richtlinien:

## 1. Fehler melden (Bugs)
Wenn du einen Fehler findest (technisch oder inhaltlich bei den Regeln), öffne bitte ein "Issue" auf GitHub oder kontaktiere mich über die [CONTACT.md](./CONTACT.md).

## 2. Neue Inhalte hinzufügen (JSON)
Wenn du neue Zauber, Gegenstände oder Regeln hinzufügen möchtest:
* Nutze immer die bestehenden JSON-Templates (siehe README oder bestehende Dateien).
* Achte auf korrekte Kommasetzung – ein fehlendes Komma macht die Datei unbrauchbar!
* Verwende für Quellenangaben immer das Buch woher du den jeweiligen Spell oder das jeweilige Item hast.

## 3. Code-Stil
* **CSS:** Nutze die Variablen aus der `style.css` (z.B. `var(--accent)`), damit das Design (Dracula-Theme) einheitlich bleibt.
* **JavaScript:** Wir nutzen Vanilla JS. Bitte füge keine schweren Frameworks oder externe Bibliotheken hinzu.

## 4. Urheberrecht
Mit dem Einreichen deiner Änderungen erklärst du dich damit einverstanden, dass dein Beitrag unter der gleichen [LICENSE](./LICENSE) wie das Hauptprojekt steht.
